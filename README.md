# Nghien-cuu-khoa-hoc
Tài liệu nghiên cứu khoa học


Nghiên cứu khoa học đại học hoa sen 2026

# 1. Tìm kiếm tài liệu & Khám phá chủ đề
Consensus: https://consensus.app (Tìm câu trả lời dựa trên bằng chứng khoa học)

ResearchRabbit: https://www.researchrabbit.ai (Tìm bài báo liên quan và hình ảnh hóa mạng lưới trích dẫn)

Connected Papers: https://www.connectedpapers.com (Vẽ biểu đồ lịch sử phát triển của một chủ đề)

# 2. Đọc hiểu & Tổng hợp dữ liệu
Elicit: https://elicit.com (Trợ lý nghiên cứu, tạo bảng matrix tổng hợp tài liệu cực tốt)

SciSpace (Typeset): https://typeset.io (Chat với PDF, giải thích công thức toán và bảng biểu)

ChatPDF: https://www.chatpdf.com (Tải PDF lên và hỏi đáp nhanh)

# 3. Hỗ trợ Viết & Chỉnh sửa (Writing Tools)
Quillbot: https://quillbot.com (Paraphrase, kiểm tra ngữ pháp và đạo văn)

Writefull: https://www.writefull.com (Chỉnh sửa ngữ pháp chuyên sâu cho văn phong học thuật)

Jenni AI: https://jenni.ai (Hỗ trợ viết tiếp câu và gợi ý trích dẫn tự động)

# 4. Phân tích dữ liệu
Julius AI: https://julius.ai (Phân tích số liệu, vẽ biểu đồ từ Excel/CSV)
